Auto-Synch

The Auto-Sync function can be activated (only when 'connected') and will fetch the current GR-55 patch settings at 5 second intervals. This is handy if making changes directly on the GR-55 and you want to see the changes reflected in the editor GUI, and handy when working with time based assign events, because the dials can be see changing in realtime.

An update may include an option to vary the sync interval times.
