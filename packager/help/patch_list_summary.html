GR-55 Patch List Summary Example

To generate a list of the patches stored within a GR-55, from the main menu of the GR-55 FloorBoard Editor, select  Tools > GR-55 Patch List Summary. A box opens with a text listing that you can select and copy for pasting into your favorite text editor, email, or wherever you like. The GR-55 stores 99 Banks of 3 Patches per Bank, for 297 User Patches. The following text is such an example, where the User memory Bank location is prefixed with 'U' and the Patch number follows the colon, both within the brackets and the Patch Name follows.
