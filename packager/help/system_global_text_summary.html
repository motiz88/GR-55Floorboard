System/Global Text Summary Example

To generate a list of System/Global settings stored within a GR-55, from the main menu of the GR-55 FloorBoard Editor, select  Tools > System/Global Text Summary. A box opens with a text listing that you can select and copy for pasting into your favorite text editor, email, or wherever you like. The following text is such an example.
