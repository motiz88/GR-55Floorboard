GR-55 FloorBoard Editor Help - INDEX

This page has links to the individual topics of the GR-55 FloorBoard Editor help pages. The GR-55 FloorBoard Editor was made for conveniently editing internal settings in the Roland GR-55 Guitar Synthesizer on most common computers. 

Preparation

To use the GR-55 FloorBoard Editor with the Roland GR-55 Guitar Synthesizer:
  • You must have a guitar or bass controller with a GK pickup mechanically installed and configured in the GR-55, see Guitar/Bass Mode GK Setup and GK Optimization.
  • You must have a MIDI connection between the computer and the GR-55, either via USB or standard DIN-5 MIDI on Mac or PC. The is no Linux MIDI driver for the GR-55 USB port, so use DIN-5.
  • You should also be familiar with what a patch is in respect to the GR-55. 
For more information on getting started with the GR-55, including controller configuration, visit the VGuitarForum.com GR-55 area topic: GR-55 TOP THINGS TO KNOW.

GLOBAL Level Parameter Settings - Affecting All Patches

NOTE: The following sections describe "global" parameters affecting all GR-55 patches. When you make a change to these parameters in the editor or in the GR-55, it is automatically written to the GR-55. Also very important to know is that any settings made in the SYSTEM, MASTER, or PDL/GK configuration will take precedence over conflicting patch level Assigns settings:
