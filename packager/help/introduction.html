Introduction to the GR-55 FloorBoard Editor

Description

The GR-55 FloorBoard Editor is MIDI editing software for the Roland GR-55 Guitar Synthesizer and is supported on specific Windows, Linux and Mac platforms. It provides GR-55 owners with a convenient user interface to edit sounds and settings within the device and to store and recall those settings on their computer for backup purposes.

Where to get the GR-55 FloorBoard Editor

To download the latest version and to learn more about the GR-55 FloorBoard Editor, visit:sourceforge.net/projects/grfloorboard/
