Welcome to GR-55 FloorBoard Help.

1.)    What is GR-55 FloorBoard?

It is a software to assist with editing parameter settings within you Roland GR-55, so your setting can be displayed graphically on a computer screen and adjusted in a simple manner.

The software also allows transferring of patch and system data to and from a computer disk file, to or from the GR-55 User patch memory locations.

2.)    What else is required to use GR-55 FloorBoard?

You need a Roland GR-55, and a computer with the GR-55 FloorBoard installed and you also need to connect the GR-55 and computer, this can be done two ways.

USB cable – for Windows and Mac OsX, a Roland GR-55 USB software driver must be downloaded from any Roland website, and installed, there is currently no driver for Linux platform.

MIDI Cables – Two midi cables and a midi device adapter which is recognised by your operating system, the device midi-out plugs to the GR-55 midi-in, and the device midi-in plugs to the GR-55 midi-out.

 

To find information on GR-55FloorBoard controls and items, press the [F1] key and click the mouse cursor on an item , and a text box with a description will appear.
